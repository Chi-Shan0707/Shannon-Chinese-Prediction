# 汉语Shannon熵研究：文献现状与研究方案

## 一、核心问题

Shannon (1951) 通过人类受试者逐字母预测英语文本，构建了条件熵链 H0→H1→H2→...→H∞：

| N（已知前N个字母） | 条件熵 (bits/letter) |
|---|---|
| 0 | 4.03 |
| 1 | ~3.6 |
| 2 | ~3.3 |
| 5 | ~2.7 |
| 10 | ~2.1 |
| 15 | ~1.9 |
| 100 | ~1.3 |
| ∞（外推） | ~1.0–1.3 |

→ 英语冗余度 ≈ 75%，每字母信息量 ≈ 1.0–1.3 bits。

**问题：汉语呢？汉语的 H0→H1→H2→...→H∞ 长什么样？**

## 二、文献现状（2026年5月核实）

### 2.1 已有成果：全部是计算法，没有预测实验

| 粒度 | 阶数 | 方法 | 熵值 | 来源 |
|---|---|---|---|---|
| 字级 | Unigram (H0) | 字频统计 | **9.62 bits/char** | 孙帆、孙茂松（《人民日报》语料库） |
| 字级 | Bigram (H1) | 条件概率 | **~7.15 bits/char** | 同上 |
| 字级 | Trigram (H2) | 条件概率 | **~6.65 bits/char** | 同上 |
| 字级 | Class-based bigram | 语义聚类 | **~4.6 bits/char** | Chang & Lin (1994) |
| 字级 | AWD-LSTM-MoS | 神经网络交叉熵 | **4.43 bpc**（原始）→ **3.96 bpc**（外推） | Takahashi & Tanaka-Ishii (2018), *Entropy* |
| 词级 | Unigram | 词频统计 | ~13.7 bits/word | — |
| 词级 | Bigram | 条件概率 | ~6.4 bits/word | — |
| 词级 | Trigram | 条件概率 | ~1.5 bits/word | — |

### 2.2 关键空白

**没有人构建过汉语的 Shannon 条件熵链。**

原因：英语只有 26 个字母，人类受试者平均猜 4-5 次能命中；汉语常用字约 6700 个，零阶平均需要猜 ~12 次，人类实验不可行。

Takahashi & Tanaka-Ishii (2018) 用 AWD-LSTM-MoS（2018 年的 LSTM，不是 Transformer）获得了最紧的汉语熵率上界 3.96 bpc，但这是**神经网络交叉熵**，不是**Shannon 式预测实验**，两者方法论不同：

| 维度 | 神经网络交叉熵 (Takahashi) | Shannon 预测实验 (本项目) |
|---|---|---|
| 方法 | 在测试集上平均 log P | 逐位置猜测 → 猜测次数分布 |
| 输出 | 一个数字（上界） | 完整的条件熵链 H0→H1→...→H∞ |
| 可解释性 | 黑箱 | 可逐级对比人类策略 |
| 与 Shannon 的可比性 | 间接 | 直接（同一方法框架） |

### 2.3 英语的再验证

Takahashi 对英语的结果：1.21 bpc（原始）→ 1.12 bpc（双变量外推），比 Shannon 的 1.3 bpc 更紧，验证了神经网络方法本身的合理性。

## 三、研究方案

### 3.1 核心思想

用 LLM **替身**受试者，对汉语做 Shannon 式逐字预测实验。

```
Shannon 1951:    人（Shannon 妻子）→ 猜英语字母 → 条件熵链
本项目:          LLM                → 猜汉字     → 条件熵链
```

### 3.2 实验设计

**预测单位**：汉字（非 token/词），直接对应 Shannon 的字母级实验。

**条件长度**：N = 0, 1, 2, 3, 5, 10, 20, 50 字。

**测试样本**：100+ 个长度为 50+ 字的现代汉语段落（覆盖新闻、文学、科普）。

**每个预测位置**：
1. 输入 Prefix（前 N 个字）
2. 获取 LLM 对下一个字的概率分布
3. 按概率排序 → 模拟"第一次猜最可能字、第二次猜第二可能字"
4. 记录猜测排名

**熵值计算**（Shannon 不等式）：
- 上界: H_上 = -Σ p_i log p_i （香农熵公式）
- 下界: H_下 = Σ i·(p_i - p_{i+1})·log i （由猜测次数分布得出）

### 3.3 技术路线

| Phase | 内容 | 规模 | 依赖 |
|---|---|---|---|
| 0 | 环境设置 | — | 确定 LLM、准备语料 |
| 1 | API 原型 | 20 样本 × 5 N 值 | GLM-4/GPT-4o API（top-K logprobs） |
| 2 | 完整实验 | 100+ 样本 × 7 N 值 | 本地 Qwen2.5（完整概率分布） |
| 3 | 分析撰写 | 图表 + 论文 | — |

### 3.4 预期结果

| N | 条件熵（汉语·预期） | 条件熵（英语·Shannon） |
|---|---|---|
| 0 | ~9.6 bits | 4.0 bits |
| 1 | ~7.2 bits | 3.6 bits |
| 2 | ~6.7 bits | 3.3 bits |
| 5 | ~5 bits | 2.7 bits |
| 10 | ~4 bits | 2.1 bits |
| 50 | ~2.5–3 bits | 1.3 bits |
| ∞ | ~2 bits？ | 1.0–1.3 bits |

### 3.5 创新点

1. **首次**构建汉语的 Shannon 条件熵链——填补了自 1951 年以来汉语信息论研究的一个基础空白。
2. 将 Shannon 实验从 26 字母扩展到 6700 汉字，用 LLM 突破了人类受试者的认知天花板。
3. 直接对比汉语 vs. 英语在同一方法论框架下的信息结构差异（每字信息量、冗余度、上下文压缩效率）。
4. 验证现代 LLM 的概率输出是否可以作为人类语言统计知识的有效代理。
