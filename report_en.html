Language, Entropy, and Chinese Texts

How Predictable Is Chinese? Replaying Shannon with a Small Language Model

A personal experiment on Chinese text predictability, internet memes, genre differences, and what happens when context gets longer.

Qwen3-0.6B Base · 11 Chinese text categories · 59,496 prediction steps

Claude Shannon once estimated the entropy of printed English by asking humans to guess the next letter of a text. That setup is charmingly simple for English letters. It becomes awkward for Chinese: the candidate set is not 26 letters, but thousands of characters, plus word-like units and idiomatic chunks. So I tried a modern substitute: let a small language model play the guessing game, record what it predicts, and ask which kinds of Chinese text are easier or harder to predict.

The strongest contribution here is not “I measured the final entropy rate of Chinese.” I did not. The stronger and more honest contribution is a reusable observation framework: a Shannon-style token-match game that makes genre differences, formulaic writing, internet meme structure, and context-length effects visible in one experiment.

What I Think the Actual Contributions Are

  1. A Chinese-friendly version of Shannon’s guessing game. Instead of asking a human to choose from thousands of Chinese characters, the experiment uses a language model as an automatic predictor and searches its top-1000 next-token candidates for the true continuation.
  2. A genre map of Chinese predictability. Encyclopedic and news-style texts are easier for the model; dialect-rich fiction and highly literary prose are harder. This is intuitive, but the experiment makes it quantitative.
  3. A useful anomaly: formulaic style is not the same as low entropy. Romance of the Three Kingdoms has the best top-1000 match rate, but it is not the lowest-entropy text. Formulaic language can help the correct token enter the candidate set without making the whole distribution extremely sharp.
  4. A small probe into meme predictability. The model nails fixed collocations such as “疯狂 → 星期四”, but punchline-like continuations such as “吃 → 肯德基” are much less stable. That difference is a nice way to separate local statistical collocation from cultural or pragmatic expectation.
  5. A context-length curve. When context grows from a few characters to about 200 characters, uncertainty falls, but with diminishing returns. This is the most Shannon-like part of the project.
  6. A useful negative result about measurement. The numbers here are model next-token entropy in nats/step. They should not be over-sold as character-level bits/char entropy rates. Making that distinction explicit is part of the value of the experiment.

If I were turning this into a paper, I would need a more careful character-level probability aggregation method. For a personal website post, though, this version is already interesting: it shows how far a small local language model can go as a microscope for Chinese textual regularity.

The Game: Token-Match Prediction

The experiment walks through a text from left to right. At each position, the model sees the prefix and computes a next-token probability distribution. I then check whether any of the top-1000 tokens is a prefix of the remaining original text. The first matching token counts as a hit; its rank, probability, and the full distribution entropy are recorded. If none of the top-1000 candidates matches, the experiment records a miss and advances by one character.

In notation, the main entropy value is the model distribution entropy

\[H(q)=-\sum_t q(t)\log q(t).\]

Because the code uses natural logarithms, the unit is nats per prediction step. This is important. These values are not directly comparable to character-level results such as bits/character. They are still very useful for comparing texts within the same model and the same experimental setup.

The Texts

I used 11 categories, ranging from Wikipedia and news to classical fiction, wuxia, modern literature, dialect fiction, internet twists, and KFC Thursday memes. The point was not to build a balanced corpus. The point was to put very different Chinese registers into the same prediction machine and see which differences survive.

Result 1: Encyclopedic Chinese Is Easy, Dialect Fiction Is Hard

The ranking makes sense. Wikipedia and news are built from conventional expository patterns: named entities, definitions, reporting formulas, and topic-specific collocations. Dialect-rich fiction is messier. It has more local voice, more spoken rhythm, more idiosyncratic phrasing, and probably less coverage in the model’s training distribution.

What I like about this result is that it turns a literary intuition into a measurable curve. “方言味儿” and “口语感” are not just stylistic impressions here; they show up as a higher-entropy prediction environment for the model.

Result 2: The Three Kingdoms Anomaly

Romance of the Three Kingdoms has the highest top-1000 match rate in the experiment: 95.0%. But it does not have the lowest mean entropy. This is the best small surprise in the project.

My interpretation: formulaic writing helps the correct continuation appear somewhere in the high-probability candidate set, but the model can still remain uncertain among several plausible continuations. Classical narrative formulas such as “却说”, “话说”, “玄德曰”, battle descriptions, titles, and repeated narrative patterns make the text guessable in a broad sense. But the probability mass is not necessarily concentrated on one token as strongly as in Wikipedia.

In other words, “the answer is in the top 1000” and “the model is very sure” are not the same statement.

Result 3: KFC Thursday Is a Collocation Machine

Chinese internet users know the structure: a normal or dramatic setup suddenly turns into “today is Crazy Thursday, V me 50.” I tested a few punchline positions directly.

The model has clearly learned “疯狂星期四” as a fixed phrase. But “eat → KFC” is much harder, because “eat” can be followed by countless food nouns. To predict KFC, the model needs not just local syntax but the meme frame.

This is why I think the meme probe is more than a cute side experiment. It separates two kinds of predictability: local collocation and culturally framed punchline expectation.

Result 4: Context Helps, But the Returns Diminish

What This Is Not

This is not a final measurement of Chinese entropy. There are three reasons.

  • The unit is nats/step, not bits/character.
  • The predictor is one small language model, not the true distribution of Chinese.
  • The game uses token matching, while classic entropy estimates are usually character-level.

A stricter version would aggregate token probabilities into character probabilities:

\[P(c\mid C)=\sum_{t:\operatorname{prefix}(t)=c} P(t\mid C).\]

That would give a proper character-level conditional entropy chain and make comparison with older bits/character estimates much cleaner. I see this post as the first pass: a way to find the interesting structure before building the stricter estimator.

Why I Still Think This Is Worth Posting

The experiment turned out to be interesting because it is not only about a number. It gives a way to ask small, interpretable questions about Chinese:

  • Which genres make a language model uncertain?
  • Does formulaic style produce low entropy or merely high candidate-set coverage?
  • Can a model distinguish fixed collocations from meme punchlines?
  • How much context is enough before the returns flatten?

Those questions are more alive than a single entropy estimate. They also fit a personal website better: this is a computational note about language texture, not a claim to have solved Chinese information theory.

Takeaway

If I had to state the contribution in one sentence, it would be this:

I built a Shannon-style prediction game that makes Chinese genre, formulaicity, meme structure, and context-length effects visible through a small language model’s next-token uncertainty.

That contribution is modest, but real. It is also extensible. The next version should add character-level probability aggregation, confidence intervals, more models, and a broader set of spoken and web-native texts. But even in this rougher form, the experiment already shows something I find beautiful: Chinese texts carry very different amounts of predictability, and a tiny local model can help us see where that predictability lives.
